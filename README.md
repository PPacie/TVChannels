# TVChannels
`tvOS` example project making use of the latest **iOS 13 Frameworks**

* `UIKit` + `SwiftUI`
* **Combine**
* **Dependency Injection** with `Storyboards` and `@IBSegueAction`.
* **MVVM** architecture pattern
* `UIViewController` transition animations.

## Requirements
- Xcode 11.3
